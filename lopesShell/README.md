# Unix/Linux Command Line Interpreter (lopeShell)

## Build
gcc -Wall -Wextra -O2 -o lopeShell lopeShell.c

## Run Interactive
./lopeShell

## Run Batch Mode
./lopeShell batch.txt

## Key Combos
Ctrl-C  → End execution (terminate running child processes)
Ctrl-\ → Exit shell

## Built-in Commands
end
quit
exit
